Дизайн-документ игры SOULRUNE

Общая концепцияНазвание проекта: SOULRUNE

Жанр: 2D-аркада и экшен на движке MonoGame

Платформа: PC (Windows)

Игровой процесс

Суть игры: Игрок управляет персонажем на арене, где должен выживать, уклоняться от летящих снарядов и атак противников. Основная фишка проекта — это полноценный бой с боссом по имени Санс под динамичную музыку.

Управление: Клавиши W / A / S / D используются для движения души. Стрелочки на клавиатуре используются для выбора действий АТАКА или ЛЕЧЕНИЕ. Клавиша Пробел используется для подтверждения выбранного действия.

Инструкция по запуску

Чтобы поиграть, сначала скачайте папку GameBuild из этого репозитория на свой компьютер. Затем откройте скачанную папку, найдите внутри файл SOULRUNE 1.exe и запустите его.
